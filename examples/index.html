Basic Optimization Example

Simple microgrid optimization with solar PV and battery storage.

from py_microgrid.tools.optimization import SystemOptimizer

# Load configuration
config = {
    'location': 'Denver, CO',
    'load_profile': 'residential',
    'pv_capacity': 300,  # kW
    'battery_capacity': 800  # kWh
}

# Initialize optimizer
optimizer = SystemOptimizer(config)

# Run optimization
results = optimizer.optimize()
print(f"LCOE: ${results.lcoe:.3f}/kWh")
print(f"Renewable fraction: {results.renewable_fraction:.1%}")

Multi-Location Analysis

Batch processing for multiple locations with different resource profiles.

from py_microgrid.tools.batch_processor import BatchProcessor

# Define multiple locations
locations = [
    {'name': 'Denver', 'lat': 39.7392, 'lon': -104.9903},
    {'name': 'Phoenix', 'lat': 33.4484, 'lon': -112.0740},
    {'name': 'Seattle', 'lat': 47.6062, 'lon': -122.3321}
]

# Initialize batch processor
processor = BatchProcessor(locations)

# Run batch optimization
results = processor.process_all()

# Compare results
for location, result in results.items():
    print(f"{location}: LCOE=${result.lcoe:.3f}/kWh")
